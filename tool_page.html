DIALOGUE EDITOR

A custom-built dialogue editing tool in UE5 developed using Slate, designed to support the creation and management of branching conversations in a structured, text-based format. This tool allows users to:


  • Create and organize multiple conversations, each composed of individual dialogue lines.
  • Add responses to any line, enabling flexible branching logic for interactive storytelling.
  • Assign a character to each line, along with a selected emotional state to reflect tone.
  • Easily navigate, edit, and expand dialogue flows through a clean, structured editor layout.

The project served as a way to deepen my understanding of Slate’s customization capabilities, especially around nested data handling, dynamic form inputs, and controlled content editing. It was designed with simplicity and clarity in mind, making it a practical solution for small-scale narrative tools, prototyping interactive dialogue, or experimenting with branching logic in a focused, text-based interface.
