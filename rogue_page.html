ROGUE-PALOMA

This project is a prototype of a roguelike game featuring procedural level generation, created in collaboration with another developer. The game dynamically creates unique dungeon layouts for each playthrough, ensuring an unpredictable experience.


For this proptotype, I developed a level editing tool in Unity to organize and store rooms used in procedural generation. The tool allows saving multiple room lists, where the room's size must be configured. For each room, three tilemap layers are stored: the background, the walls, and the areas where enemies or objects can spawn. The tool also marks the sides where rooms will connect, helping the level designer leave the necessary spaces open.


My contributions:

  • Procedural generation implementation
  • Tilemap room editor tool
  • User interface implementation
  • Level design
